# Scaffold.js App

## Getting Started

To run the application:

1. Clone the repository
2. Run `npm link`
3. Execute `create-scaffold-app your-app`

## Goals

Create a TypeScript-based, all-in-one web app framework with batteries included!

## Tech Stack

- TypeScript
- Prisma
- Hono
- Auth.js
